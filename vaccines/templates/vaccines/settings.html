💉 Step 1: Adding a Vaccine

1. Go to the Vaccines tab and click "+ Add Vaccine".
2. Enter the vaccine name (e.g., BCG, Penta, MMR).
3. Check "Live Attenuated" if this is a live vaccine. Live vaccines require a 28-day gap between each other.
4. Add Schedule Rules — one row per dose:
  • Dose # — 1 for first dose, 2 for second, etc.
  • Min Age — Earliest allowed age in days (safety floor)
  • Recommended Age — Ideal age to give the dose
  • Max Age — Latest allowed age (leave blank if no limit)
  • Min Interval — Minimum days from the previous dose (use 0 for the first dose)
💡 Tip: For first doses, set "Min Interval" to 0. For subsequent doses, set it to the minimum wait time (e.g., 28 days for a 4-week interval).

🔄 Step 2: Adding Catch-up Rules (Optional)

Catch-up rules apply when a child is behind schedule. They define how many doses are still needed and the interval to use.

1. While adding or editing a vaccine, scroll to the "Catch-up Rules" section.
2. Set the age band (From age / To age) for when this catchup rule applies.
3. Set Prior Doses — how many doses the child has already received.
4. Set Total Doses Needed — how many they should get in total to complete catch-up.
5. Set Min Interval — minimum days between catch-up doses.

🔗 Step 3: Creating Vaccine Groups

Groups are for vaccine families that share the same antigen. For example, Penta (given to infants), DTC, and Td all protect against Diphtheria, Tetanus, and Pertussis — so they belong to the "DTP Family" group.

1. Go to the Groups tab and click "+ Add Group".
2. Name the group (e.g., DTP Family).
3. Select all vaccines that belong to this group.
4. Set the Min Valid Interval — the minimum days between any two doses in this group (usually 28 days).

📋 Step 4: Adding Group Rules

Group rules tell the engine which vaccine to give next based on the child's age and how many group doses they've had.

Example: For DTP Family:
  • Child is 6 weeks old with 0 doses → Give Penta
  • Child is 18 months old with 3 doses → Give DTC (booster)
  • Child is 6 years old with 4 doses → Give Td
For each rule, set:
  • Prior Doses — how many valid group doses the child has
  • From Age / To Age — age bracket in days
  • Vaccine to Give — which vaccine to administer
  • Min Interval — minimum days from the last group dose
💡 Tip: Think of group rules as a decision table: "IF the child has X doses AND is between age A and B, THEN give Vaccine Y with at least N days since the last dose."

⚠️ Common Mistakes to Avoid

1. Forgetting to add vaccines to a group before creating group rules — add the vaccine first, then create the group.
2. Overlapping age ranges in group rules — make sure each age bracket is distinct for the same number of prior doses.
3. Setting Min Interval to 0 for non-first doses — always set a valid interval (e.g., 28 days) for doses after the first.
4. Not creating both schedule rules AND group rules — grouped vaccines still need schedule rules for the engine to calculate target dates.
